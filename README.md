## 📋 Cursos — Ordem de Estudo Recomendada

> Os cursos estão organizados do **mais fundamental ao mais avançado**, seguindo uma trilha de aprendizado progressiva.

---

### 1️⃣ Algoritmos e Lógica de Programação — O Curso COMPLETO
**Instrutor:** Nélio Alves  
**Por que começar aqui?** É a base de tudo. Antes de qualquer linguagem, é preciso entender como pensar de forma lógica e estruturada. Este curso ensina exatamente isso, de forma independente de linguagem.

🔗 [Acessar curso](https://www.udemy.com/course/curso-algoritmos-logica-de-programacao/)

---

### 2️⃣ Linguagem C — Iniciante ao Avançado
**Instrutor:** One Day Code  
**Por que em segundo?** C é a base histórica e técnica de praticamente todas as linguagens modernas. Aprender C consolida o raciocínio de baixo nível, ponteiros, memória e estruturas, tornando qualquer linguagem seguinte muito mais fácil de absorver.

🔗 [Acessar curso](https://www.udemy.com/course/curso-completo-de-linguagem-c-iniciante-ao-avancado/)

---

### 3️⃣ Python 3 — Do Básico ao Avançado (com projetos reais)
**Instrutor:** Luiz Otávio Miranda  
**Por que em terceiro?** Com a lógica e os fundamentos de C no bolso, Python se torna muito mais rápido de aprender. É a linguagem mais versátil da lista — serve para back-end, automação, ciência de dados e segurança.

🔗 [Acessar curso](https://www.udemy.com/course/python-3-do-zero-ao-avancado/)

---

### 4️⃣ Desenvolvimento Web Completo
**Instrutor:** Equipe do curso  
**Por que em quarto?** Com Python em mãos, é hora de entrar na web. Este curso cobre o front-end (HTML, CSS, JavaScript) e o back-end completo, conectando tudo ao banco de dados. É o ponto de virada para se tornar um desenvolvedor web.

🔗 [Acessar curso](https://www.udemy.com/course/curso-web/)

---

### 5️⃣ Java COMPLETO — Programação Orientada a Objetos + Projetos
**Instrutor:** Nélio Alves  
**Por que em quinto?** Java aprofunda o paradigma de Orientação a Objetos de forma muito sólida. Quem já tem Python e lógica bem fundamentados aproveita Java para amadurecer o pensamento sobre arquitetura e design de sistemas.

🔗 [Acessar curso](https://www.udemy.com/course/java-curso-completo/)

---

### 6️⃣ Python Para Hackers
**Instrutor:** BurnSec — Ethical Hacking  
**Por que por último?** Este é o curso mais avançado da trilha. Para aproveitar ao máximo o conteúdo de segurança ofensiva e desenvolvimento de ferramentas para pentest, é essencial já ter uma base sólida em Python, lógica e web. É o ponto mais avançado da jornada.

🔗 [Acessar curso](https://www.udemy.com/course/burnsec-python-para-hackers/)
